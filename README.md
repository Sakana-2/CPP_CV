# Simple Photo Edditing App
Studying Image information processing only using OpenCV Matrix

## What this can do
* median blur
* gaussian blur
* Sobel filter
* Prewitt filter
* Laplacian
* threshold
* adaptive threshold
* unsharpmask
* dithering
* etc...
